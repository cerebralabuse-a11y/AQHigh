# AQHigh - Air Quality & Cigarette Counter

Check your local air quality and see how many cigarettes you've effectively smoked just by breathing. A satirical take on air pollution awareness.

## Technology Stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To run this project locally:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Deployment

Build the project for production:

```sh
npm run build
```
